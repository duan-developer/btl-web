# Node Login App

Ứng dụng web đơn giản viết bằng Node.js và Express hỗ trợ đăng ký/đăng nhập với session.

## Cài đặt

```bash
npm install
npm run dev
```

Ứng dụng sẽ chạy tại `http://localhost:3000`. Tài khoản mẫu: `demo` / `demo123`.

> Lưu ý: dữ liệu người dùng được lưu tạm thời trong bộ nhớ và sẽ mất khi khởi động lại server. Hãy thay thế bằng cơ sở dữ liệu thực tế nếu dùng trong môi trường sản xuất.
